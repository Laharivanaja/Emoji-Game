# 😀 Emoji Game

An interactive React-based Emoji Memory Game where users must click unique emojis without repeating any selection.

---

## 🚀 Features

- Click on emojis to score points
- Score increases for every unique click
- Game resets if the same emoji is clicked twice
- Top score tracking
- Responsive UI
- Randomized emoji positions after every click

---

## 🛠️ Tech Stack

- React 17
- React Router (if used)
- JavaScript (ES6+)
- CSS3
- Create React App

## 📦 Installation

Clone the repository:


git clone https://github.com/Laharivanaja/Emoji-Game.git


Navigate into the project:


cd Emoji-Game


Install dependencies:


npm install


Start the development server:


npm start
